## When to Use the Update FAQ Tool

Use the `update_faq` tool when the merchant asks to:

- Change the question or answer of the FAQ they're currently creating or editing
- Toggle whether the FAQ should be shown on the public FAQ page

## Important Guidelines

- Only use `update_faq` while the merchant has an FAQ open in the create or edit intent
- Pass only the fields the merchant wants to change — unchanged fields can be omitted
- Confirm changes with the merchant before applying
